Infinitely scalable data storage for object, block and file types.

Ceph provides a flexible, scalable, reliable and intelligent solution for data storage, regardless of data type.

Ceph does more than store data, and provides effective self-management and innovative algorithms to secure, manage and protect data. From unifying various data types in one unified RADOS cluster to allowing infinite scalability, Ceph is intelligent and innovative data technology.

Store Data in the Cloud

Ceph allows for hybrid-cloud data storage, with reliable and secure management. Organizations can take their applications into the future, whether on-prem, or in the public or private cloud.

Whatever delivery framework you require, Ceph can be adapted and applied accordingly.

Object storage

For businesses requiring seamless access to objects, Ceph provide comprehensive storage compatible with popular interfaces including Amazon S3.

CRUSH Algorithm

The CRUSH (Controlled Replication Under Scalable Hashing) algorithm keeps organizations’ data safe and storage scalable through automatic replication. Using the CRUSH algorithm, Ceph’s storage clusters are liberated from scalability and performance limitations. By replicating and rebalancing data within the cluster automatically, the CRUSH algorithm alleviates administrants of tedious tasks.

Reliable Autonomic Distributed Object Store (RADOS)

Through RADOS, organizations can store and access object, block and file types in one unified storage cluster, making data and monitoring easy to manage. Each one of your applications can use the object, block or file system interfaces to the same RADOS cluster simultaneously, which means your Ceph storage system serves as a flexible foundation for all of your data storage needs. Additionally, Ceph’s RADOS provides you with extraordinary data storage scalability, able to store petabytes to exabytes of data, using widely aware hardware.
